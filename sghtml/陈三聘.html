陈三聘

朝代: 宋代

诗歌

三登乐

形式: 词

标签: 宋代, 陈三聘

一品归来， 
强健日、小园幽圃。 
扁舟兴、恐天未许。 
想当年、持汉节， 
众齐咻楚。 
丹忠此日， 
盛名千古。 
掞词章、师海内， 
纬文经武。 
莫寒盟、故山旧侣。 
到鲈乡、还又是， 
秋风斜雨。 
鸣刀鲙雪， 
未应便去。
临江仙

形式: 词

标签: 宋代, 陈三聘

白首故人重会面， 
论交尔汝忘形。 
从今心迹喜双清。 
飞鸿追往事， 
为蝶笑馀生。 
琥珀杯浓春正好， 
此怀端为君倾。 
旧时猿鹤敢寒盟。 
鸠居从拙计， 
鹏翼任高程。
减字木兰花

形式: 词

标签: 宋代, 陈三聘

先生困熟。 
万卷书中聊托宿。 
似怯清寒。 
更爇都梁向博山。 
游仙梦杳。 
啼鸟声中春又晓。 
未著乌纱。 
独坐溪亭数落花。
千秋岁

形式: 词

标签: 宋代, 陈三聘

当年渔隐， 
路转桃溪汇。 
流水下， 
青山外。 
客行花径曲， 
月上松门对。 
撑艇子， 
雪中蓑笠亲曾载。 
老去谁倾盖。 
腰瘦频移带。 
人健否， 
花仍在。 
明年春更好， 
来向花前醉。 
青鬓改， 
恁时难拼千金买。
南柯子·南歌子

形式: 词

标签: 宋代, 陈三聘

月傍云头吐， 
风将雨脚吹。 
夜深乌鹊向南飞。 
应是星娥颦恨、入双眉。 
旧怨垂千古， 
新欢只片时。 
一年屈指数佳期。 
到得佳期别了、又相思。
卜算子

形式: 词

标签: 宋代, 陈三聘

雪後竹枝风， 
醉梦风吹醒。 
瘦立寒阶满地春， 
淡月梅花影。 
门外辘轳寒， 
晓汲喧金井。 
长笛何人更倚楼， 
玉指风前冷。
好事近

形式: 词

标签: 宋代, 陈三聘

枝上几多春， 
数点不融香雪。 
纵有笔头千字， 
也难夸清绝。 
艳桃秾李敢争妍， 
清怨笛中咽。 
试策短筇溪上， 
看影浮波月。
如梦令

形式: 词

标签: 宋代, 陈三聘

珍重故人相许。 
来向水亭幽处。 
文字间金钗， 
消尽晚天微暑。 
无雨。 
无雨。 
不比寻常端午。
宜男草

形式: 未知

标签: 宋代, 陈三聘

绿水黏天净无浪。 
转东风、縠纹微涨。 
个中趣、莫遣人知， 
容我日日、扁舟独往。 
平生书癖已无恙。 
解名缰、更逃羁网。 
春近也、梅柳频看， 
枝上玉蕊、金丝暗长。
念奴娇

形式: 词

标签: 宋代, 陈三聘

扁舟此计， 
问当年、谁与寻盟鸥鸟。 
许国勋名彝鼎在， 
风月不妨吟笑。 
碧草台边， 
红云溪上， 
寿杖扶诗老。 
水浮天处， 
未应俗驾曾到。 
盛事埒美知章， 
鉴湖君赐， 
宸翰今题号。 
指点飞烟轻霭外， 
有路直通仙岛。 
蓑笠渔船， 
琴书客坐， 
清夜尊罍倒。 
未须归去， 
片蟾初上林表。
惜分飞

形式: 词

标签: 宋代, 陈三聘

莫唱骊驹容首聚。 
花径重来微步。 
从此朝天去。 
故山怨鹤栖猿侣。 
试卜西园春在否。 
无奈濛濛细雨。 
明日长亭路。 
断魂芳草人何处。
朝中措

形式: 词

标签: 宋代, 陈三聘

去年曾醉杏花坊。 
柳色间轻黄。 
重觅旧时行迹， 
春风满路梅香。 
平沙岸草， 
夫差故国， 
知是吾乡。 
梦断数声柔橹， 
只应已过横塘。
梦玉人引

形式: 词

标签: 宋代, 陈三聘

别来何处， 
酒醒後， 
梦难觅。 
晚日溪亭， 
清晓便挂帆席。 
满载离愁， 
指去程、还作江南行客。 
目断层城， 
数迢迢山驿。 
素巾空染， 
泪痕斑、应是暗中滴。 
记得轻分， 
玉箫犹自凄咽。 
昨夜东风， 
梅柳惊春色。 
料伊也、没心情， 
过却好天良夕。
水调歌头

形式: 词

标签: 宋代, 陈三聘

玉鉴十分满， 
清露一年秋。 
漂流踪迹， 
谁念楚尾与吴头。 
此夜刮明尘眼。 
望极好张诗胆。 
何处有高楼。 
浩荡银潢冷， 
缥缈白云浮。 
笑劳生， 
难坎止， 
亦乘流。 
阑干拍碎， 
清夜起舞不胜愁。 
万里关河依旧。 
一寸功名乌有。 
清泪滴衣裘。 
老去心空在， 
归梦绕苹洲。
浣溪沙

形式: 五律

标签: 宋代, 陈三聘

帘押低垂月影疏。 
梅枝和雪玉相扶。 
儿家春信入来无。 
半坠宝钗慵览镜， 
任偏罗髻却拈书。 
琴心谁与问相如。
浪淘沙

形式: 七律

标签: 宋代, 陈三聘

风雨晚春天。 
芳兴慵悭。 
浅红稠绿满园间。 
独有梨花三四朵， 
留住春寒。 
年少跃金鞍。 
咫尺关山。 
倦飞如我已知还。 
洒向东风千点泪， 
衣上重看。
满江红

形式: 词

标签: 宋代, 陈三聘

绀縠浮空， 
山拥髻、晚来风急。 
吹骤雨、藕花千柄， 
艳妆新浥。 
窥鉴粉光犹有泪， 
凌波罗袜何曾湿。 
讶汉宫、朝罢玉皇归， 
凝情立。 
尊前恨， 
歌三叠。 
身外事， 
轻飞叶。 
怅当年空击， 
誓江孤楫。 
云色远连平野尽， 
夕阳偏傍疏林入。 
看月明、冷浸碧琉璃， 
君须吸。
秦楼月·忆秦娥

形式: 未知

标签: 宋代, 陈三聘

青楼缺。 
楼心人待黄昏月。 
黄昏月。 
入帘无奈， 
柳绵吹雪。 
谁人弄笛声呜咽。 
伤春未解丁香结。 
丁香结。 
鳞鸿何处， 
路遥江阔。
菩萨蛮

形式: 词

标签: 宋代, 陈三聘

筇枝探得梅开了。 
青鞋渐踏江头草。 
日日作东风。 
海棠相次红。 
离多良会少。 
此计应须早。 
莫待作行人。 
却将愁送春。
虞美人

形式: 词

标签: 宋代, 陈三聘

融融睡觉东风息。 
行到溪亭侧。 
一枝梅玉似人人。 
索笑依然消瘦、不禁春。 
相逢试问情多少。 
应怪山翁老。 
翠罗高护结花邻。 
一任馀芳争学、捧心颦。
蝶恋花

形式: 词

标签: 宋代, 陈三聘

阊阖城西山四面。 
鸭绿鳞鳞， 
轻拍横塘岸。 
一阵东风羊角转。 
望中已觉孤帆远。 
独恨寻芳来较晚。 
柘老桑稠， 
农务村村遍。 
山鸟劝酤官酒贱。 
炊烟深巷听缫茧。
西江月

形式: 词

标签: 宋代, 陈三聘

诗眼曾逢花面， 
画图还识春娇。 
当年风格太妖饶。 
粉腻酥柔更好。 
酒晕不温香脸， 
玉慵犹怯轻绡。 
春风别後又秋高。 
再见只应人老。
鹊桥仙

形式: 词

标签: 宋代, 陈三聘

银潢仙仗， 
离多会少， 
朝暮世情休妒。 
夜深风露洒然秋， 
又莫是、轻分泪雨。 
云收雾散， 
漏残更尽， 
遥想双星情绪。 
凭谁批敕诉天公， 
待留住、今宵休去。
鹧鸪天

形式: 七律

标签: 宋代, 陈三聘

翦碎霜绡巧作团。 
玉纤特地破朝寒。 
疏花好向钗横见， 
瘦影难敲月堕看。 
将旧恨， 
入眉弯。 
不须多样缕金旛。 
当时千点东风泪， 
怪见妆成粉未乾。
