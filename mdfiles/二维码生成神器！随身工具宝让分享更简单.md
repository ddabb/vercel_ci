# 二维码生成神器！随身工具宝让分享更简单

## 分享的烦恼

生活中经常需要分享信息：

- **分享链接**：发一大串网址给朋友，又长又容易输错
- **分享WiFi**：密码太长记不住，一个个数字念太麻烦
- **分享名片**：纸质名片容易丢，信息更新了也没法同步
- **分享文本**：大段文字复制粘贴太麻烦

## 随身工具宝二维码生成器

「随身工具宝」小程序的二维码生成器，让分享变得简单：

### 多种内容类型支持

1. **网址链接**：把网页链接转成二维码，扫码即打开
2. **纯文本**：长段文字一键转码，扫码即可查看
3. **名片信息**：姓名、电话、邮箱一键生成电子名片
4. **WiFi信息**：输入SSID和密码，扫码自动连接
5. **电话/短信**：一键生成拨打电话或发送短信的二维码

### 自定义功能

- **颜色自定义**：前景色、背景色自由搭配
- **Logo添加**：可以添加自定义Logo到二维码中心
- **尺寸调整**：根据需要调整二维码大小
- **容错级别**：支持不同级别容错，保证扫码成功率

### 使用方法

1. 选择要生成的内容类型
2. 输入相关信息
3. 自定义样式（可选）
4. 生成二维码，长按保存或分享

## 二维码小贴士

### 二维码的优势
- **信息容量大**：可以存储大量信息
- **容错能力强**：部分损坏仍能识别
- **使用方便**：手机一扫即可获取信息

### 使用注意事项
- 不要在二维码中存储敏感信息
- 定期更新二维码内容（如联系方式变更）
- 选择合适的容错级别，保证远距离也能识别
