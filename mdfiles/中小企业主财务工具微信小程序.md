# 中小企业主必看！微信小程序帮你管理公司财务

## 中小企业主的困境

小公司请不起专业会计：

- 老板自己兼会计
- 每天忙于算账
- 现金流算不清楚
- 决策缺乏数据支持

但有些财务问题，自己必须搞清楚：
- 这个月赚了多少？
- 成本有哪些？
- 下个月能收回多少钱？

## 创业者必须懂的财务

1. **利润计算**
- 收入 - 成本 = 利润
- 毛利率、净利率
- 这个月是亏是赚

2. **现金流**
- 应收账款何时到账
- 应付账款何时支付
- 资金链紧不紧

3. **成本核算**
- 人工成本
- 房租、水电
- 营销费用

4. **投资决策**
- 设备要不要更新？
- 新项目能不能投？
- 回报周期多长？
