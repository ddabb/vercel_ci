# 健康瘦身必看！BMI计算器帮你科学管理体重

## 被忽视的身体指标

很多人关心体重，但更应该关心的是BMI。这个看似简单的数字，其实藏着健康的秘密：

- **体重轻不等于健康**：有人体重正常，体脂却超标
- **体重重不等于肥胖**：健身人群肌肉量大，体重数字并不低
- **BMI超标但不在意**：内脏脂肪堆积，增加心血管风险

BMI（身体质量指数）是国际上常用的衡量人体胖瘦程度以及是否健康的一个标准。

## BMI的计算公式

BMI = 体重(kg) ÷ 身高(m)的平方

例如：一个70kg的人，身高1.75米
BMI = 70 ÷ (1.75)² = 22.86

## BMI的评判标准

根据中国标准：
- **BMI < 18.5**：体重过轻
- **18.5 ≤ BMI < 24**：正常范围
- **24 ≤ BMI < 28**：超重
- **BMI ≥ 28**：肥胖

## 不仅仅看BMI

BMI虽好，但也有局限：
- 不区分肌肉和脂肪
- 不适合未成年人
- 不适合运动员

建议结合腰围、体脂率等指标综合判断。
